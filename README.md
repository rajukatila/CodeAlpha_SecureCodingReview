# Secure Coding Review

## Overview
This project demonstrates a Secure Coding Review by analyzing a vulnerable login application and implementing security improvements.

## Vulnerabilities Identified
1. Hardcoded Credentials
2. Plaintext Password Storage
3. Lack of Input Validation
4. No Brute Force Protection

## Secure Coding Improvements
- Password Hashing using SHA-256
- Improved Authentication Logic
- Better Credential Handling
